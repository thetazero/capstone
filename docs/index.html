Finding Eigenvalues of the linearized Navier Stokes and Euler Equations

Euler Case

{{Context that I don't quite understand goes here}}

The problem is to find a positive solution to the following equation.

$$\frac{\lambda}{\rho_{0}}+f(\lambda)+g(\lambda)=0$$

Where \(f(\lambda)\), \(g(\lambda)\) are defined by the following continued fractions

$$f(\lambda)=\frac{1}{\frac{\lambda}{\rho_{1}}+\frac{1}{\frac{\lambda}{\rho_{2}}+\dots}},\space \space g(\lambda)=\frac{1}{\frac{\lambda}{\rho_{-1}}+\frac{1}{\frac{\lambda}{\rho_{-2}}+\dots}}$$

Where \(\rho_{n}\) is defined by the following formula

$$\rho_{n}=1-\frac{\|p\|^2(1-\alpha^2\|p\|^2)}{\|q+np\|^2(1-\alpha^2\|q+np\|^2)}$$

However the first equation only works for case I0.

Navier-Stokes Case

{{More context that I don't quite get}}

For this case, the method for solving is mostly identical. There is a very similar equation to solve:

$$a_{0}(\lambda, \nu)+f(\lambda, \nu)+g(\lambda, \nu)=0$$

Where \(f(\lambda)\), \(g(\lambda)\) are defined by the following continued fractions

$$f(\lambda)=\frac{1}{a_{1}(\lambda, \nu)+\frac{1}{a_{2}(\lambda, \nu)+\dots}},\space \space g(\lambda)=\frac{1}{a_{-1}(\lambda, \nu)+\frac{1}{a_{-2}(\lambda, \nu)+\dots}}$$

Where

$$a_{n}(\lambda, \nu)=\frac{\lambda+\nu\|q+np\|^2}{\rho_{n}}$$

Computationally, this leads to much faster convergance than in the Euler Case because of the large constant terms.
